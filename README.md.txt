# Kenya Spatial Land Portal

A web-based GIS application for land parcel visualization and proximity analysis.

## Features
- Interactive map with land parcels
- Proximity analysis (2km radius)
- Infrastructure layers (schools, health centers, towns)
- Search functionality by plot ID

## Setup
1. Clone repository
2. Run `npm install`
3. Set up PostgreSQL database with `sql.sql`
4. Configure environment variables
5. Run `npm start`

## Technologies
- Node.js/Express
- PostgreSQL/PostGIS
- Leaflet.js
